********** Carte Cadeau Interactive **********

## Description
Une page web interactive et amusante présentant une carte cadeau.
 L'utilisateur peut interagir avec la boîte pour réclamer son cadeau, ce qui déclenche une belle animation de chute de neige. Il est également possible de fermer la boîte, qui réapparaît de manière ludique après quelques secondes.

## Fonctionnalités
* **Animation de neige :** Génération dynamique de flocons de neige via JavaScript avec des délais et positions aléatoires.
* **UI Moderne :** Interface stylisée avec CSS utilisant Flexbox, des dégradés, des ombres portées et des transitions fluides.
* **Interactivité :** Boutons animés et gestion des événements de clic pour masquer/afficher les éléments.

## Technologies Utilisées
* **HTML5** pour la structure.
* **CSS3** pour le design et les animations (keyframes, transitions).
* **JavaScript ** pour la logique d'interaction et la génération des éléments du DOM.
