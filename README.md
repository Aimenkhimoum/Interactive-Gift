# 🎁 Carte Cadeau Interactive

**Auteur :** KHIMOUM Aimen Mohamed  
*Créé avec passion*

---

## Description
Ce projet est une page web interactive et immersive conçue pour offrir une expérience ludique de déballage de cadeau. 

L'utilisateur peut interagir avec une boîte cadeau pour réclamer son contenu. Cette action déclenche une **animation dynamique de chute de neige**. Pour ajouter une touche d'amusement, la boîte peut être fermée et réapparaît de manière surprenante après quelques secondes.

##  Fonctionnalités
* **❄️ Animation de neige :** Génération procédurale de flocons de neige via JavaScript, avec des délais, des tailles et des positions aléatoires pour un effet naturel.
* ** UI Moderne & Élégante :** Interface entièrement stylisée en CSS utilisant **Flexbox**, des dégradés de couleurs (gradients), des ombres portées (box-shadows) et des transitions fluides.
* ** Interactivité avancée :** * Boutons animés au survol et au clic.
    * Gestion dynamique du DOM pour masquer ou afficher les éléments sans rechargement de page.
    * Effet de "respawn" ludique de la boîte cadeau.

## Technologies Utilisées
* **HTML5** : Structuration sémantique du contenu.
* **CSS3** : Design responsive, animations complexes (`keyframes`) et transitions.
* **JavaScript (ES6+)** : Logique d'interaction, gestion des événements de clic et moteur de génération des flocons.

---

###  Aperçu du fonctionnement
> L'interaction principale repose sur l'écoute d'événements JavaScript qui modifient les classes CSS pour déclencher les animations de sortie et l'apparition de la neige en arrière-plan.
